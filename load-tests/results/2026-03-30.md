## Execution Prerequisites

To run these tests, a live environment is needed:

```sh
# 1. Start services
docker compose up -d

# 2. Run migrations
pnpm --filter @agentops/db migrate

# 3. Start server
pnpm --filter @agentops/server dev
# Server will be at http://localhost:4000

# 4. Seed test user
curl -s -X POST http://localhost:4000/api/auth/register \
  -H 'Content-Type: application/json' \
  -d '{
    "companyName":"loadtest-co",
    "companyDisplayName":"LoadTest Co",
    "email":"loadtest@agentops.example.com",
    "name":"Load Test User",
    "password":"LoadTestPassword123!"
  }'

# 5. Create test department + workspace, export IDs
# 6. Create test agent, export API key as K6_AGENT_KEY
```

Then run all tests:
```sh
K6_BASE_URL=http://localhost:4000 \
K6_AGENT_KEY=ak_<hex> \
K6_WORKSPACE_ID=<uuid> \
k6 run load-tests/auth-flow.js

k6 run load-tests/agent-checkin.js
k6 run load-tests/file-upload.js
k6 run load-tests/websocket.js
```
